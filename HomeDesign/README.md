A visionOS demo built with [UntoldEngine](https://github.com/untoldengine/UntoldEngine) that showcases
mixed-reality spatial interaction: users calibrate a miniature floor plan onto their real floor, walk
it up to full scale, and decorate rooms by placing furniture from a catalog with floor/wall placement,
snap-to-grid, two-hand scale/rotate, and drag gestures.
